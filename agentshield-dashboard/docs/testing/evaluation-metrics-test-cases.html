📘 About This Document This document provides comprehensive, repeatable test cases for the AgentShield Evaluation Metrics module. The module implements a Three-Layer Evaluation Architecture (Node / Session / System) using LLM-as-a-Judge with Chain-of-Thought grading, automated Persona-Driven Simulation, and a confidence-based Human-in-the-Loop (HITL) review workflow. Tests can be executed via the Admin Dashboard UI or the REST API.
