01 Budget Management (CRUD)

A Budget defines token and cost limits for a specific scope (user, team, department, agent, project, or global). Each budget has a period (daily/weekly/monthly/quarterly), warning threshold, and optional hard limit that blocks requests when exceeded.

Scope Types & Hard Limits
ScopeDescription
userLimit applies to a specific user ID
teamLimit applies to a team (e.g., "engineering")
departmentLimit applies to a department
agentLimit applies to a specific agent UUID
projectLimit applies to a project identifier
globalOrganization-wide limit (scope_id = "global")

02 Model Pricing Management (CRUD)

The Model Pricing tab allows admins to manage LLM cost rates stored in the model_pricing database table. Pricing is used to auto-estimate cost when upstream agents don't return cost data. The pricing cache refreshes every 5 minutes.

Key Design Points
  • Database-driven — pricing lives in the model_pricing table, not hardcoded
  • Vendor tracking — each model has an explicit vendor (OpenAI, Anthropic, Google, etc.)
  • Cached — in-memory cache with 5-minute TTL for performance
  • Admin CRUD — add, edit, delete models via UI or API

05 Gateway Budget Enforcement

When a budget is set to hard limit and is exceeded, the gateway middleware blocks requests. When the budget is a soft limit, requests proceed but a warning is logged.

Agent Request → Auth Check → Guardrails → Budget Checker → │ │ ┌── Budget OK ─────────────────────── ✓ Forward to Agent │ ├── Soft limit exceeded ──────── ⚠ Log warning, forward │ └── Hard limit exceeded ──────── ✗ 429 Too Many Requests

06 Token Tracking & Estimation

The gateway now always tracks tokens for every agent call. When the agent returns real token counts, those are used. When it doesn't (e.g., MCP agents), tokens are estimated from payload size (~4 chars per token). Records are flagged with is_estimated.

Token Estimation Logic
ScenarioBehavior
Agent returns usage.input_tokensReal tokens used, is_estimated = false
Agent returns no usage dataEstimated from payload: Math.ceil(text.length / 4), is_estimated = true
MCP agent callExtracts from callResult._meta.usage if available, else estimates
